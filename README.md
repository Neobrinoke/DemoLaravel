# Demo Laravel
- `composer install`
- Configuration de votre BDD & du .env
- `php artisan migrate`
- `php artisan serve`
- Rendez-vous sur l'url `localhost:8000`

## Les liens d'accès
 - http://localhost:8000/list - Liste des tâches
 - http://localhost:8000/create-task - Création d'une nouvelle tâche
